# Typography
<Subtitle>
version {version}
</Subtitle>

This package exposes values for typography-related css, such as font sizes and font weights. The exports available are the following:

### size

```js
{
  bodySmall: '0.875rem',
  bodyMedium: '1rem',
  bodyLarge: '1.125rem',
  h1: '3rem',
  h2: '2rem',
  h3: '1.5rem',
  h4: '1.125rem',
}
```

### weight

```js
{
  normal: 300,
  bold: 500,
}
```

## Usage
* To install `Typography` package with npm run terminal:

```jsx
npm i @pm-kit/typography

```
* Import `Typography` in your componenent to render

```jsx
import {size, weight} from '@pm-kit/typography'

```
